Prettier

https://prettier/marketplace

Prettier is a code formatter that can help you maintain consistent code formatting across your project. With Prettier, you don't need to worry about manually formatting your code - simply write your code and let Prettier handle the rest. Prettier supports many programming languages and integrates with Visual Studio Code and Atom seamlessly.

GitLens

https://gitlens/marketplace

GitLens is a powerful Git integration tool for Visual Studio Code. With GitLens, you can see Git blame annotations and commit details directly in your code editor, view branch and tag details, and much more. GitLens also includes a number of powerful search and filtering tools that make it easy to navigate even the largest Git repositories. If you use Git for version control, GitLens is definitely worth checking out.
